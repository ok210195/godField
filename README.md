# GodField共同開発

ゴッドフィールドをリスペクトした、Webベースのカード対戦ゲーム開発プロジェクトです。
HTML、CSS、JSを使用したリアルタイムな対戦を再現します。

## 🚀 セットアップ方法

1. リポジトリをクローン
   　　　git clone [リポジトリのURL]

2. VScode内のターミナルに以下のコマンドを入力する
   　　　npm install
   　　　npm start
   

   📂 フォルダ構成
server.js: バックエンド(審判的な役割を果たしてくれる)

cards.csv: カードデータ（名前、属性、攻撃力など）

index.html: ゲーム画面

main.js: クライアントサイドの挙動

style.css: UIの変更が可能

🛠 開発ルール (GitHub Flow)

mainブランチに直接pushしない。
